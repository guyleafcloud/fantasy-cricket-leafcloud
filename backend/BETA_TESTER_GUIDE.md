### Batting Points (Tiered System)
- **Runs 1-30:** 1.0 point per run
- **Runs 31-49:** 1.25 points per run
- **Runs 50-99:** 1.5 points per run
- **Runs 100+:** 1.75 points per run
- **Strike Rate Bonus:** Points Ã— (SR / 100)
- **50 runs:** +8 bonus points
- **100 runs:** +16 bonus points
- **Duck (out for 0):** -2 points

### Bowling Points (Tiered System)
- **Wickets 1-2:** 15 points each
- **Wickets 3-4:** 20 points each
- **Wickets 5-10:** 30 points each
- **Maiden over:** 15 points
- **Economy Rate Bonus:** Points Ã— (6.0 / ER)
- **5 wicket haul:** +8 bonus points

### Fielding Points
- **Catch:** 15 points (30 for wicket-keeper)
- **Stumping:** 15 points
- **Run Out:** 6 points

### Player Multiplier System
Each player has a **performance handicap multiplier** (0.69 to 5.0):
- **Lower multiplier (0.69):** Best real-life players - points are REDUCED (handicap)
- **Higher multiplier (5.0):** Weaker players - points are BOOSTED
- **This balances the game** - makes all players viable picks!

Multipliers adjust weekly based on performance (Â±15% drift).
